<h1 align="center"> SISTEMA DE FACTURACIÓN </h1>
¡Di adiós a la facturación manual y tediosa! Nuestro programa transforma tus facturas a mano en archivos PDF profesionales en menos de 10 minutos, y de forma automática, registra todas tus ventas en una hoja de Excel organizada. Digitaliza, optimiza y gana tiempo.
<br>
<br>

> [!IMPORTANT]
> Es importante que coloque este programa en su propia carpeta. De esta forma, los archivos PDF y Excel generados se guardarán ordenadamente en el mismo lugar.

<br>
<br>

> [!IMPORTANT]
> Para ejecutar nuestro programa sin problema sigue estos pasos dependiendo tu IDE:

# Spyder
1. instala las siguientes librerias en la terminal propia de Spyder:
<br>

   ```
    conda install -c conda-forge reportlab
   ```
